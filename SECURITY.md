# Security Policy

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Instead, report privately via
[GitHub Security Advisories](https://github.com/ECgear/wise-pdf/security/advisories/new) (preferred),
or the contact listed on the [ECgear GitHub profile](https://github.com/ECgear).

Please include steps to reproduce and the affected version. We aim to acknowledge
reports within a few days and will keep you updated on the fix.

Note that wise pdf runs entirely in the browser and never uploads your files, so
most data-exposure risks do not apply; reports about the processing code, build
pipeline, or dependencies are still very welcome.

## Supported versions

The latest released version receives security fixes.
